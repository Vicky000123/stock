# 如何使用
1. 下載xml及exe檔
2. 打開xml檔案
3. 輸入欲查詢股票代碼至stocknumber欄位裡
4. 輸入時間範圍，共四項（startyear,startmonth,endyear,endmonth)
5. 在excelname欄位輸入檔案名稱
6. 存檔並將xml關閉
7. 執行exe
